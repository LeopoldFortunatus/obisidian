# Сборка транскриптов.
## Для шульман:
```
Сделай подробный конспект выпуска [укажи название программы, выпуск, дату, если есть] по транскрипту. Структурируй по темам, а не по хронологии. Дай смысловую выжимку, оформи связным повествованием. Избегай маркированных списков. Не пиши итогов и заключений. Убери всё рекламное и второстепенное, если оно не связано с основной темой. Сохрани стиль деловой, ясный и нейтральный.

Пример структуры, которая мне подходит:

1. Вступление и анонсы
    
2. Основные политические/социальные события
    
3. Законодательные изменения
    
4. Международная повестка
    
5. Концептуальные/теоретические блоки
    
6. Ответы на вопросы слушателей
```

